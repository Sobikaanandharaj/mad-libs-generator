# Mad Libs Generator 🎉

A fun, beginner-friendly Python program that generates silly random stories using words provided by the user — without them knowing the actual story in advance!

## Features
- 10 different story templates
- A random story is chosen each time you run the program
- Takes user input for place, adjective, animal, food, verb, and number
- Fills in the blanks to generate a unique, funny story every time

## How It Works
1. The user is asked to enter a few random words (a place, an adjective, an animal, a food, a verb, and a number) — without knowing how they'll be used
2. The program randomly selects one story template from a list of 10
3. The chosen words are inserted into the blanks of the selected story
4. The final, often hilarious, story is displayed on screen
